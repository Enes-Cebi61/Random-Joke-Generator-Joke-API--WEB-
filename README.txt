# Web Site that Fetches Random Jokes using Joke API

This project is a front-end web site developed using HTML, CSS, and JavaScript. The site fetches a random joke using a joke API and displays it to the user.

## How to Run

1. Clone this repository.
2. Open the index.html file in a web browser.

## Technologies Used

- HTML
- CSS
- JavaScript
